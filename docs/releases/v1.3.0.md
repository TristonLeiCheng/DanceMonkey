# DanceMonkey v1.3.0

## 新功能

- **定时提醒模块**：将喝水、久坐提醒整合为统一定时提醒，支持用户自定义提醒与多种重复规则（间隔、连续使用、每日、每周、每月、一次性）
- **多种桌面弹窗样式**：磨砂卡片、圆形弹窗、灵动岛（顶部胶囊 + 点击展开）、Toast 通知、顶部横幅、紧凑胶囊
- **通知方式可选**：桌面弹窗或宠物气泡；单条提醒可覆盖全局弹窗样式
- **截屏磨砂背景**：桌面弹窗支持真实屏幕模糊（可在定时提醒页开关）
- **导入 / 导出**：提醒配置可导出为 JSON，支持合并或替换导入

## New features

- **Scheduled reminders**: unified water, sedentary, and custom reminders with flexible recurrence
- **Six desktop popup styles**: glass card, circular, Dynamic Island, toast, top banner, compact pill
- **Pet bubble or desktop popup** notifications; per-reminder style override
- **Screen-capture frosted glass** backgrounds for desktop popups
- **Import / export** reminder configurations (JSON)

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.0.zip`
2. 解压到任意**本地目录**（请勿直接从网络路径运行）
3. 运行 `DanceMonkey.exe`（需 Windows 10/11 与 WebView2 运行时）

## Install

1. Download `DanceMonkey-win-x64-1.3.0.zip`
2. Extract to a **local folder** (do not run directly from a network share)
3. Run `DanceMonkey.exe` (requires Windows 10/11 and WebView2 Runtime)
