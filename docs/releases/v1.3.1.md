# DanceMonkey v1.3.1

## 修复

- **升级安装路径**：从带版本号的解压目录（如 `DanceMonkey-win-x64-1.3.0`）升级时，自动迁移到固定目录 `%LOCALAPPDATA%\DanceMonkey\app`，避免文件夹名与程序版本不一致
- **开机启动**：升级后若已启用开机启动，注册表路径会自动更新到新安装目录
- **升级提示**：从版本化目录升级时，设置页会提示将迁移到固定目录

## Fixes

- **Update install path**: upgrading from a versioned extract folder (e.g. `DanceMonkey-win-x64-1.3.0`) now migrates to `%LOCALAPPDATA%\DanceMonkey\app` so the folder name stays consistent with the app version
- **Startup entry**: registry run key is updated after upgrade when startup is enabled
- **Upgrade UI**: shows migration target when upgrading from a versioned folder

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.1.zip`
2. 解压到本地目录，或双击 `启动 DanceMonkey.bat` 安装到固定目录
3. 运行 `DanceMonkey.exe`（需 Windows 10/11 与 WebView2 运行时）

## Install

1. Download `DanceMonkey-win-x64-1.3.1.zip`
2. Extract locally, or run `启动 DanceMonkey.bat` to install to the canonical app folder
3. Run `DanceMonkey.exe` (Windows 10/11 + WebView2 Runtime required)
