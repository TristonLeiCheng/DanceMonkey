# DanceMonkey v1.3.10

## 改进

- **会议中心 UI 紧凑化**：工作台、会议库等页面采用更紧凑的布局与控件样式
- **工作台行动项 / 决策项**：会中可直接添加行动项与决策记录，支持负责人、截止日期；行动项可一键同步到 ZenTask
- **全局行动项 Tab**：跨会议汇总行动项，按状态（进行中 / 已完成 / 未同步）与项目筛选
- **全局决策项 Tab**：跨会议汇总决策记录，按项目筛选
- **决策数据模型**：新增 `DecisionItems`（含决策人），自动迁移旧版 `Decisions` 字符串列表
- **ZenTask 回写**：行动项同步后记录 `ZenTaskId`，避免重复写入
- **会议库**：支持「显示归档」开关

## Improvements

- **Compact meeting hub UI**
- **Workbench action items & decisions** with ZenTask sync
- **Global action items / decisions tabs** with filters
- **DecisionItems model** with legacy migration
- **ZenTaskId** tracking on synced action items
- **Show archived** toggle in meeting library

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.10.zip`
2. 解压后运行 `DanceMonkey.exe`（升级会在当前目录就地替换）
3. 从网络盘启动请使用 **`启动 DanceMonkey.bat`**
4. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.10.zip`
2. Run `DanceMonkey.exe` (updates apply in the same folder)
3. Use **`启动 DanceMonkey.bat`** when launching from a UNC path
