# DanceMonkey v1.3.11

## 改进

- **会议中心 Web 化重构**：WPF 多 Tab 界面改为 WebView2 + `meeting-hub.html`，统一导航与视觉风格
- **MeetingHubService 后端**：业务逻辑集中到 C# 服务层，通过 WebMessage 与前端双向通信
- **功能完整保留**：工作台、日历、会议库、行动项、决策项、例会、模板、统计等模块均迁移至新 UI
- **交互增强**：文本选中浮动工具栏、纪要预览弹窗、Toast 提示、响应式三栏工作台布局

## Improvements

- **Meeting hub Web UI**: WebView2 + HTML replaces large WPF XAML surface
- **MeetingHubService**: centralized backend with WebMessage bridge
- **All meeting modules** preserved in the new shell
- **Richer UX**: selection bubble, summary modal, toasts, responsive workbench

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.11.zip`
2. 解压后运行 `DanceMonkey.exe`（升级会在当前目录就地替换）
3. 从网络盘启动请使用 **`启动 DanceMonkey.bat`**
4. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.11.zip`
2. Run `DanceMonkey.exe` (updates apply in the same folder)
3. Use **`启动 DanceMonkey.bat`** when launching from a UNC path
4. Windows 10/11 + WebView2 Runtime required
