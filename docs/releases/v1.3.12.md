# DanceMonkey v1.3.12

## 改进

- **结构化议程**：议题改为「标题 + 正文」双字段，支持每议题下记录讨论内容；自动兼容旧版字符串议程
- **工作台布局优化**：会议信息栏可折叠；三栏比例 26/34/40，宽屏下信息栏收起后自动让位给议程与手记
- **会议库详情**：Markdown 纪要渲染为 HTML 预览（非编辑模式）
- **UI 细节**：议程块样式、图标按钮、纪要 Markdown 样式、响应式间距调整

## Improvements

- **Structured agenda items** (title + body) with legacy string migration
- **Collapsible meeting info** panel and improved workbench grid layout
- **Library detail** Markdown HTML preview
- **UI polish** for agenda blocks, icon buttons, and spacing

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.12.zip`
2. 解压后运行 `DanceMonkey.exe`（升级会在当前目录就地替换）
3. 从网络盘启动请使用 **`启动 DanceMonkey.bat`**
4. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.12.zip`
2. Run `DanceMonkey.exe` (updates apply in the same folder)
3. Use **`启动 DanceMonkey.bat`** when launching from a UNC path
4. Windows 10/11 + WebView2 Runtime required
