# DanceMonkey v1.3.2

## 变更

- **移除个人主页**：从侧栏与代码中移除局域网发布个人主页功能（含编辑器、HTTP 服务与相关资源）
- **Codex 自动配置**：启动 DM Proxy 时可自动写入 `~/.codex/config.toml`、`auth.json` 与用户环境变量（`OPENAI_API_KEY`、本地代理 bypass），便于 Codex CLI 直连本机中转
- **DM Proxy 工具调用**：改进 Responses API → Chat Completions 转换，合并连续 `function_call`、正确处理嵌入 `tool_calls` 的消息

## Changes

- **Remove personal homepage**: sidebar entry and LAN publishing feature removed from the app
- **Codex auto-config**: when DM Proxy starts, optionally writes `~/.codex/config.toml`, `auth.json`, and user env vars for Codex CLI
- **DM Proxy tool calls**: better Responses → Chat Completions mapping for batched function calls and embedded tool_calls

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.2.zip`
2. 解压到本地目录，或双击 `启动 DanceMonkey.bat` 安装到固定目录
3. 运行 `DanceMonkey.exe`（需 Windows 10/11 与 WebView2 运行时）

## Install

1. Download `DanceMonkey-win-x64-1.3.2.zip`
2. Extract locally, or run `启动 DanceMonkey.bat` to install to the canonical app folder
3. Run `DanceMonkey.exe` (Windows 10/11 + WebView2 Runtime required)
