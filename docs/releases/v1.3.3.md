# DanceMonkey v1.3.3

## 修复

- **升级后无法启动**：从带版本号的解压目录或网络盘（UNC）双击 `DanceMonkey.exe` 可能无反应；启动时会自动跳转到本机固定目录 `%LOCALAPPDATA%\DanceMonkey\app`
- **升级迁移提示**：应用内升级迁移目录后，会在旧目录生成 `启动 DanceMonkey.bat` 与说明文件，并弹窗提示新路径
- **避免降级覆盖**：若本机固定目录已有较新版本，从旧解压目录启动时会直接打开新版本，不会用旧文件覆盖

## Fixes

- **Post-update launch**: double-clicking from a versioned extract folder or UNC path may silently fail; bootstrap now relaunches from `%LOCALAPPDATA%\DanceMonkey\app`
- **Migration hints**: in-app upgrade writes a launcher batch file and shows the new install path
- **No downgrade**: launching from an older extract folder opens the newer canonical install when available

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.3.zip`
2. 解压后优先双击 **`启动 DanceMonkey.bat`**，或直接运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.3.zip`
2. Prefer **`启动 DanceMonkey.bat`**, or run `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. Windows 10/11 + WebView2 Runtime required
