# DanceMonkey v1.3.4

## 改进

- **DM Proxy / Codex**：清理未配对的 `function_call` 与缺少 tool 回复的 `tool_calls`，避免上游 Chat Completions 返回 400
- **Codex 自动配置**：在 `config.toml` 中写入 `supports_websockets = false`（DM Proxy 仅 HTTP），减少 Codex CLI 走 WebSocket 失败
- **Codex 代理页**：说明文案更新，提示需重启 Codex 后生效

## Improvements

- **DM Proxy / Codex**: sanitize orphaned function_call / incomplete tool_calls chains to prevent upstream 400 errors
- **Codex auto-config**: set `supports_websockets = false` in `config.toml` (DM Proxy is HTTP-only)
- **Codex proxy UI**: updated hint text about WebSocket and restart

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.4.zip`
2. 解压后双击 **`启动 DanceMonkey.bat`**，或运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.4.zip`
2. Run **`启动 DanceMonkey.bat`** or `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. Windows 10/11 + WebView2 Runtime required
