# DanceMonkey v1.3.5

## 改进

- **Codex 默认 API 模式**：自动配置写入 `forced_login_method = "api"`、`cli_auth_credentials_store = "file"`，强制使用 API Key 而非 ChatGPT 登录
- **环境变量**：同时设置 `OPENAI_BASE_URL` 与 `OPENAI_API_KEY`，便于 Codex / VS Code 扩展直连 DM Proxy
- **auth.json 清理**：仅保留 `OPENAI_API_KEY`，移除 ChatGPT OAuth 会话字段，避免仍弹出登录页
- **DM Proxy 页**：新增「立即写入 Codex 配置」按钮，无需启动代理也可应用配置

## Improvements

- **Codex API-by-default**: auto-config sets `forced_login_method = "api"` and file-based credentials
- **Env vars**: writes both `OPENAI_BASE_URL` and `OPENAI_API_KEY`
- **auth.json**: API key only, clears ChatGPT OAuth tokens
- **DM Proxy UI**: "Apply Codex config now" button

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.5.zip`
2. 解压后双击 **`启动 DanceMonkey.bat`**，或运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.5.zip`
2. Run **`启动 DanceMonkey.bat`** or `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. Windows 10/11 + WebView2 Runtime required
