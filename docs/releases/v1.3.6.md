# DanceMonkey v1.3.6

## 改进

- **DM Proxy Codex 配置重构**：改用 `model_provider = "api"` + `[model_providers.api]`，`requires_openai_auth = false`，不再写入 `OPENAI_BASE_URL` / `OPENAI_API_KEY` 或 `auth.json`
- **Codex 模型预设**：下拉可选 gpt-5.5、Claude Opus/Sonnet 4.6–4.8，也支持自定义输入
- **更多 config.toml 字段**：上下文窗口、自动压缩阈值、推理强度（含 xhigh）、推理摘要
- **NO_PROXY 一键设置**：新增「设置 NO_PROXY」按钮，等价于 `setx NO_PROXY "localhost,127.0.0.1"`，便于绕过企业代理访问本机代理
- **立即写入配置**：无需勾选自动配置即可写入 Codex `config.toml`

## Improvements

- **Codex via model_provider**: writes `model_provider = "api"` with local DM Proxy provider block; no auth.json / OPENAI env vars
- **Model presets**: gpt-5.5 and Claude Opus/Sonnet 4.6–4.8, editable combo
- **Extra config.toml fields**: context window, auto-compact limit, reasoning effort/summary
- **NO_PROXY button**: one-click `setx NO_PROXY "localhost,127.0.0.1"`
- **Apply config anytime**: write config.toml without enabling auto-config on proxy start

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.6.zip`
2. 解压后双击 **`启动 DanceMonkey.bat`**，或运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时
