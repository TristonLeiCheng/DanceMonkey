# DanceMonkey v1.3.7

## 改进

- **统一模型配置**：中转配置区的「模型」同时用于 DM Proxy 上游与 Codex `config.toml`，不再重复配置
- **模型预设下拉**：gpt-5.5、Claude Opus/Sonnet 4.6–4.8 预设移至中转配置，支持自定义输入
- **默认模型与推理**：默认 `gpt-5.5`，推理强度默认 `medium`
- **恢复 Codex 默认**：可一键移除 `config.toml` 中 DanceMonkey 注入的 DM Proxy 配置块

## Improvements

- **Single model setting** for proxy upstream and Codex `config.toml`
- **Model presets** in relay config section (gpt-5.5, Claude Opus/Sonnet 4.6–4.8)
- **Defaults**: `gpt-5.5` model, `medium` reasoning effort
- **Restore Codex defaults** button to remove managed DM Proxy block from `config.toml`

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.7.zip`
2. 解压后双击 **`启动 DanceMonkey.bat`**，或运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.7.zip`
2. Run **`启动 DanceMonkey.bat`** or `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. Windows 10/11 + WebView2 Runtime required
