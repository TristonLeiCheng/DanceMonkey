# DanceMonkey v1.3.8

## 改进

- **会议中心工作台重构**：左侧集中编辑会议信息（主题、状态、模板、项目、时间、标签、参会人），右侧议程 / 手记 / AI 纪要三栏布局
- **新建与打开**：工作台支持「新建会议」与「从会议库打开」，会议库详情可一键跳回工作台编辑
- **会议库增强**：全文搜索（标题、标签、参会人、议程、手记、纪要）；列表 / 按项目分组两种视图
- **纪要就地编辑**：会议库详情支持编辑并保存纪要 Markdown
- **Tab 导航优化**：标签页横向滚动换行，避免多 Tab 被裁切

## Improvements

- **Meeting workbench redesign**: metadata sidebar + agenda / notes / AI summary columns
- **New / open from library** flow between workbench and meeting library
- **Library search** and flat vs project-grouped tree view
- **In-place summary editing** in library detail
- **Tab bar** horizontal scroll with wrap layout

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.8.zip`
2. 解压后双击 **`启动 DanceMonkey.bat`**，或运行 `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
3. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.8.zip`
2. Run **`启动 DanceMonkey.bat`** or `%LOCALAPPDATA%\DanceMonkey\app\DanceMonkey.exe`
