# DanceMonkey v1.3.9

## 改进

- **在线升级就地更新**：应用内升级在当前运行目录直接替换文件，不再迁移到 `%LOCALAPPDATA%\DanceMonkey\app`，避免电脑里出现两份安装
- **启动引导调整**：仅 UNC 网络路径仍复制到本机固定目录；本地磁盘（含带版本号的解压文件夹）不再自动迁移
- **会议中心**：会议归档（Markdown 移至 `_archive`）、永久删除会议及文件、自定义模板可删除
- **模板管理**：内置/自定义模板分区展示，支持编辑与删除自定义模板

## Improvements

- **In-place updates**: upgrades overwrite the current install directory instead of migrating to a fixed path
- **Bootstrap**: UNC-only relaunch to local app dir; local folders stay put
- **Meeting hub**: archive meetings, delete with Markdown cleanup, delete custom templates

## 安装

1. 下载 `DanceMonkey-win-x64-1.3.9.zip`
2. 解压后运行 `DanceMonkey.exe`（升级会在当前目录就地替换）
3. 从网络盘启动请使用 **`启动 DanceMonkey.bat`**
4. 需 Windows 10/11 与 WebView2 运行时

## Install

1. Download `DanceMonkey-win-x64-1.3.9.zip`
2. Run `DanceMonkey.exe` (updates apply in the same folder)
3. Use **`启动 DanceMonkey.bat`** when launching from a UNC path
4. Windows 10/11 + WebView2 Runtime required
